Doom Emulator

Experience the classic Doom game directly in your browser!

About the Emulator

This project uses js-dos to emulate the original Doom experience right in your browser. The setup was created using the bun create dosbox template, making it efficient and modern.

How It Works

  • js-dos: A powerful JavaScript-based DOS emulator that allows classic games to run in the browser.
  • Bun: The project uses bun for managing the runtime and build process, ensuring top-notch performance.
